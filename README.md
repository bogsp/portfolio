# Personal Portfolio

A minimalist portfolio site showcasing full-stack development and product management work.

## Features

- **Monochrome Design**: Clean, distraction-free interface with dark mode support
- **Portfolio Showcase**: PM Dashboard and Developer Documentation projects
- **Experience Timeline**: 26 years of cross-domain expertise
- **Blog-Ready**: Structure in place for future technical writing

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Shadcn UI
- Vercel deployment

## Local Development
```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed on Vercel with automatic deployments from main branch.
